標記代碼部分非常簡單： 
view code
只有一點點的標記代碼。 當你將一個片段標記為parallel，Lift將這個片段的計算交給線程池裡的其他線程。 同時，Lift繼續計算頁面的其他部分，並將最後的計算結果重新組合起來，再將組合后的頁面發送給瀏覽器。 讓我們來看看snippet代碼：
